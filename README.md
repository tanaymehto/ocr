# Flask OCR Web App - Quick Start

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Set Google Cloud credentials:
```bash
set GOOGLE_APPLICATION_CREDENTIALS=path\to\your\credentials.json
```

3. Run the app:
```bash
python app.py
```

4. Open browser: http://localhost:5000

---

## Requirements

- Tesseract installed: C:\Program Files\Tesseract-OCR\tesseract.exe
- Google Cloud Vision API credentials (JSON file)
- Python 3.8+

---

## Quick Test

1. Upload a scanned document (PNG/JPG)
2. Select OCR engine
3. Click "Extract Text"
4. See results instantly

Done!
